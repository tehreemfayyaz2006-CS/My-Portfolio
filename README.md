### 🔹 Electricity Bill Generator ⚡

📌 Description:
A Generating Electricity Bill System is a software or process used to calculate and produce monthly electricity bills for consumers based on their electricity usage.

🎯 Purpose:

The main purpose of this system is to:
* Automate the billing process
* Calculate accurate electricity charges based on units consumed
* Reduce manual errors in bill generation
* Save time and improve efficiency
* Provide transparent billing information to customers

 📌 How it Works:
 * Meter Reading Input
* The system takes the current and previous meter readings of a customer.
* Unit Calculation
Electricity units are calculated using the formula:
* Units Consumed = Current Reading − Previous Reading
